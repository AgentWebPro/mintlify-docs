<AccordionGroup>
  <Accordion title="How do I reset my password?">
    Use the **Reset password** link on the [login page](https://app.agentweb.pro). There is currently no password reset option in Settings.
  </Accordion>
  <Accordion title="Is the engagement time on the SEO/GEO page average or total?">
    Total. The value is cumulative time all visitors spent actively engaged on the page during the selected date range, sourced from Google Analytics 4's `userEngagementDuration` metric.

    Example: if 100 users each spent 30 seconds, the displayed time is approximately 50 minutes.

    To estimate average engagement per visit, divide by the page's session count for the same period.
  </Accordion>
  <Accordion title="Where can I see what's been generated?">
    - Posts, drafts, images, videos, and landing pages → [Creative](/user-guide/creative).
    - Reports and analyses → [Performance → Research Reports](/user-guide/performance#research-reports).
    - One-off Agent Mode conversations → Agent Mode chat history.
  </Accordion>
  <Accordion title="Why is Emma's output generic?">
    Almost always a context problem. Check that:

    - Brand Guidelines and Market Strategy are filled out completely.
    - Knowledge Base has examples of strong content in the format you want.
    - You gave Emma a clear objective, audience, and channel in the prompt.

    See [Agent Mode → Best practices](/user-guide/agent-mode#best-practices) for the patterns that produce sharp output.
  </Accordion>
  <Accordion title="Can I use Emma outside the portal?">
    Yes. Emma is available in Claude.ai, Claude Code, and OpenClaw via MCP. See [Connect Emma](/playbooks/connect-emma).
  </Accordion>
  <Accordion title="How do I add team members?">
    See [Organization sharing](/onboarding/organization-sharing). Short version: **Settings → Organization → Members → Invite member**.
  </Accordion>
  <Accordion title="Something is broken or behaving wrong">
    Type **File bug** in any Agent Mode conversation and describe what you saw. The team triages bugs, and once the fix ships, Emma's behavior updates automatically. See [Agent Mode → Report a bug or request a fix](/user-guide/agent-mode#report-a-bug-or-request-a-fix).
  </Accordion>
</AccordionGroup>
